İki Kişilik Yüz Yapboz Oyunu

Bu proje, OpenCV ve MediaPipe kullanılarak geliştirilen gerçek zamanlı iki kişilik etkileşimli bir görüntü işleme oyunudur.

Oyunda oyuncuların yüzleri kamera aracılığıyla algılanır ve yapboz parçalarına ayrılır. Oyuncular, el hareketlerini kullanarak parçaları sürükleyip doğru yerlere yerleştirerek karşı tarafın yüzünü tamamlamaya çalışır.

Kullanılan Teknolojiler
Python
OpenCV
MediaPipe
NumPy
Proje Özellikleri
Gerçek zamanlı yüz algılama
El takibi ve hareket analizi
El hareketleri ile sürükle-bırak sistemi
İki kişilik oyun sistemi
Glow efekti
Trail (hareket izi) efekti
Geri sayım animasyonu
Kazanma animasyonu
Konfeti efekti
Tam ekran oyun deneyimi
Çalışma Mantığı
Oyuncular sırayla yüzlerini kameraya gösterir.
Sistem yüzleri algılar ve yapboz parçalarına ayırır.
Yapboz parçaları karıştırılır.
Oyuncular el hareketleri ile parçaları taşıyarak yapbozu tamamlamaya çalışır.
En kısa sürede tamamlayan oyuncu kazanır.
Amaç

Bu proje, görüntü işleme, el takibi, yüz algılama ve insan-bilgisayar etkileşimi teknolojilerini kullanarak eğlenceli ve etkileşimli bir oyun sistemi geliştirmek amacıyla hazırlanmıştır.

Kurulum

Gerekli kütüphaneleri yüklemek için:

pip install opencv-python mediapipe numpy

Projeyi çalıştırmak için:

python yuz_yapboz.py
